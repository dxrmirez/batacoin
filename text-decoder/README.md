# text-decoder
Es un proyecto sencillo para poner a prueba mis conocimientos en JS/CSS, acompañado de un diseño UI/UX agradable y funcional.

---
Las "llaves" de encriptación que utilicé son las siguientes:

`La letra "a" es convertida a "nap"`

`La letra "e" es convertida a "scre"`

`La letra "i" es convertida a "ciry"`

`La letra "o" es convertida a "zork"`

`La letra "u" es convertida a "pulm"`

#### Features:
* Funciona solo con letras minúsculas y sin acentos
* Capaz de convertir una palabra encriptada a su versión original
* El resultado se muestra en la pantalla
* Un botón que copia el texto encriptado/desencriptado al portapapeles, osea lo mismo que `crtl+c`
